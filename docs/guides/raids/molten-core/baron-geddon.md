The "Living Bomb" and the source of the *Binding of the Windseeker*.

* **Living Bomb:** If you get this, **RUN** to the designated empty area. You will explode and fly into the air. (Priests: Shield the bomb target so they don't die from fall damage).
* **Ignite Mana:** Drains mana over time. Priests/Paladins must dispel.
* **Inferno:** Geddon stays still and pulses massive AoE fire damage. **Melee: Run out!**
