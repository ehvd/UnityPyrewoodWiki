The "Add Management" fight.

* **Firesworn Adds:** Warlocks must **Banish** as many as possible (usually 4-5).
* **Separation:** Tanks pull the un-banished adds away from Garr. When an add dies, it knocks everyone back.
* **Magma Shackles:** Massive slow. Dispel magic from your casters.
