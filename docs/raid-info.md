# 📋 Raid Information & Guidelines

Welcome to the central hub for our raid requirements, schedules, and loot systems. Please ensure you are familiar with
these rules before joining a raid.

## 🛠️ Addon Requirements

### Mandatory (Must have to participate)
* **DBM / BigWigs:** Essential for encounter timers and mechanic alerts.
* **Threatmeter (ClassicThreatMeter):** Crucial for managing aggro.
* **Core Loot Manager (CLM):** **Mandatory** for the EPGP system and rolling on items.
* **LootReserve / Gargul:** (For Lootmasters) Manages reserves and rolls.
* **BijouRR:** (For Lootmasters in ZG) Streamlines token distribution.

### Highly Advised (Recommended)
* **GTFO:** Audio alerts for standing in "fire."
* **PallyPower:** For Paladin blessing management.
* **Decursive:** For Healers/Hybrids to simplify dispelling.
* **Tranq Rotation Addon:** Required for Hunters on specific bosses.
* **ItemRack:** For quick gear swaps between encounters.
* **ThreatPlates:** Recommended for Tanks to track nameplate aggro.

## 💰 Loot System: EPGP
We use the **EPGP (Effort Points / Gear Points)** system to ensure fair distribution based on participation.

### How it works:
* **PR (Priority):** Calculated as $PR = EP \div GP$.
* **Highest PR wins the loot.**
* **Decay:** A **25% decay** is applied to both EP and GP every week to prevent point hoarding.
* **Points:** You only earn EP for bosses you are present for. 
* **Bench:** If asked to play an alt for the raid, your main is "benched" and still receives EP.

### Roll Brackets:
| Bracket | Cost | Description |
| :--- | :--- | :--- |
| **Main Spec (MS)** | 100% GP | Primary gear upgrades. |
| **Minor Upgrade (MU)** | 50% GP | Small stat increments. |
| **Off Spec (OS)** | 25% GP | Gear for secondary roles. |

!!! warning 

    **Anti-Exploitation:** Players repeatedly skipping upgrades to "save PR" for BiS items may be denied loot by a soft
    loot council (GM/Officers). Please respect the honor system.
